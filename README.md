# ShortLinks

Lightweight and extensible **URL shortener for .NET**.

## Features
- Create short links
- Resolve & redirect
- SQL Server / EF Core support
- Expiration support
- Hit tracking
- Controller-based (no ABP)

## Installation

```bash
dotnet add package ShortLinks.Core
dotnet add package ShortLinks.Storage.EFCore
dotnet add package Microsoft.EntityFrameworkCore.SqlServer
